Goal:
Improve poster/product image presentation across homepage, New arrivals, shop grid, and product page while keeping mockup hover fade working smoothly.

Current situation:

New requirement:
Add a controlled poster card presentation mode instead of hard-replacing the current behavior.

Suggested presentation modes:

1. current / fallback

   * Keep the current setup exactly as it works today.
   * This must remain available until I approve the new setup.

2. full-image

   * The artwork fills the image area of the card.
   * Product title and price stay below the image area.
   * This should feel like a clean ecommerce card:

     * image on top
     * text below
     * consistent card width
   * Choose the best fit behavior to keep this visually balanced across photo-like posters, portrait posters, square posters, and landscape posters.
   * Avoid obviously bad cropping.

3. stage

   * A fixed neutral/warm background stage.
   * The poster artwork is centered inside the stage, keeps its real aspect ratio, and gets a subtle shadow.
   * Similar in principle to modern poster stores where the product tile is consistent but the poster shape remains visible.
   * The poster should be large enough to compensate for the reduced usable area:

     * suggested max-height around 86–90%
     * suggested max-width around 82–88%
   * Avoid making the poster look tiny or like accidental object-contain.

Hover fade requirement:

* Fix the mockup hover fade so it works both when moving the mouse onto the artwork and when moving the mouse away.
* Right now the fade only feels smooth on hover-out.
* The hover-in should also blend smoothly into the mockup.
* Preferred implementation:

  * If a hover mockup exists, always render both the base artwork layer and the hover mockup layer.
  * Do not mount/render the hover mockup only after hover starts.
  * Base artwork layer:

    * opacity-100 normally
    * opacity-0 on group-hover
  * Hover mockup layer:

    * opacity-0 normally
    * opacity-100 on group-hover
  * Both layers should use the same transition:

    * transition-opacity
    * duration-600
    * ease-out
* The hover mockup should be an absolute overlay over the whole image stage.
* The hover mockup should use object-cover and fill the stage.
* This must work in all poster presentation modes:

  * current/fallback
  * full-image
  * stage
* There should be no flicker, no layout shift, and no instant snap on hover-in.
