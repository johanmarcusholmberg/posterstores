Core public rule:
A mockup is public-eligible only when:

* mockupImageUrl exists
* status is not failed
* template is active
* for gallery: isGallery = true
* for primary: isPrimary = true
* for hover: isHoverMockup = true
* for AI-rendered mockups: approvedForPublic = true

If a mockup row is marked primary/hover/gallery but has no mockupImageUrl:

* do not use template background
* do not use template preview
* do not use live composite
* fall back to the poster’s normal image
