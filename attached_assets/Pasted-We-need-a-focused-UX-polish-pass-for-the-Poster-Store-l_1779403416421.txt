Improve the poster listing UX so the store feels cleaner, more premium, and easier to browse.

Requested changes:

1. Clean header with expandable search

  - PostersOfSpain logo on the left
  - Existing icons on the right: favorites, cart, account, plus a search icon

2. Listing page title logic

3. Improve active filter display
Current behavior lists selected filters as chips, which becomes messy when many filters are active.

Change active filter display to a compact summary:
- Show a maximum of 2 selected filter chips.
- If more than 2 filters are selected, show a compact “+X more” item after the first two chips.
- Keep “Clear all” visible when any filter/search is active.
- Clicking individual visible chips should still remove that filter if current functionality supports it.
- If implementing a popover/drawer for “+X more” is simple, make it show all active filters with remove actions.
- If that is too much for this pass, “+X more” can be non-interactive or simple text, but structure the code so it can be upgraded later.
- Make sure this looks good on mobile and does not wrap awkwardly.

Example:
Valencia ×   Andalusia ×   +3 more   Clear all
