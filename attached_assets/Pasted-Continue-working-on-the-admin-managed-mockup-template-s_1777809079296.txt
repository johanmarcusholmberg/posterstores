Please check whether this should be changed to timestamp with time zone / timestamptz to avoid timezone confusion.

Preferred:
- Use timestamptz for detected_at
- Store detection timestamps consistently in UTC
- Make sure frontend displays the timestamp correctly if shown
- Add a safe migration if needed

2. End-to-end mockup preview check

Verify the full flow:

Admin:
- Upload mockup image
- AI detects placement
- Admin adjusts placement manually
- Admin saves mockup

Frontend/product:
- Active mockup templates are fetched from backend
- Inactive mockups are not shown to normal users
- Poster image is placed using saved x/y/width/height
- Manually adjusted placement values are respected
- Fallback placement values are respected
- Featured mockup is used first if available
- If no compatible mockups exist, the UI falls back gracefully

A 50x50 poster should only show square-compatible mockups.

If needed, add helper logic for:
